### **Fitness Hub – Web Application**
## **Description**

  Fitness Hub is a responsive fitness website built using HTML, CSS, and JavaScript.
  It includes a virtual assistant that works with voice commands (like “open leg workout” or “open back workout”) and redirects users to the correct workout page.
  It also has a chatbot to answer basic fitness-related questions and user doubts.

## Features

  Responsive and user-friendly design
  
  Workout pages (leg, back, chest, etc.)
  
  Voice-controlled virtual assistant for navigation
  
  Chatbot for fitness-related queries
  
  Interactive UI using JavaScript
  
  Clean layout using HTML and CSS

## Technologies Used

  HTML
  
  CSS
  
  JavaScript
  
  Web Speech API (for voice assistant)
  
  Visual Studio

## How to Run the Project

  Download or clone the repository.
  
  Open the project folder.
  
  Open index.html in your browser.
